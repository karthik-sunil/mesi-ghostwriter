The following files are included in this folder:
1. Analysis.xlsx: this contains the analysis for the coherence traffic and the NoC energy consumption
2. Coherence.ipynb: this contains the python script that generated the permutations and combinations for the coherence traffic analysis
3. Ghostwriter.m: the murphi file for formally verifying the protocol
4. Ghostwriter.out: the murphi output for the protocol
5. ghostwriterGEM5.isa: this contains the modifications to the ISA to include the Scribble instructions
6. ghostwriterGEM5.sm: this contains the modifications to the cache coherence protocol to include the additional Gs and Gi states
